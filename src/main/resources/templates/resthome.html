REST Home Page

Actions of resources

Http Method Resource EndPoint Description JSON
GET /questionnaires Gets all questionnaires /questionnaires
GET /questionnaires/id Gets questionnaire by id /questionnaires/4
GET /answers/questionnaireId Gets all answers by questionnaire id (text, checkbox or radio) /answers/4
GET /answers/questionnaireId/questionId Gets all answers by questionnaire id and question id (text, checkbox or radio) /answers/4/9
POST /answers/questionnaireId Adds new answer
